README.md

YP-Spur (YamabicoProject-Spur)

Purpose

YP-Spur is a mobile robot motion control software with coordinate frame based commands.

Getting Started

Learn to use YP-Spur at the documentation pages.

Install

$ mkdir build
$ cd build
$ cmake ..
$ make
$ sudo make install

YP-Spur (YamabicoProject-Spur)

Purpose

YP-Spur is a mobile robot motion control software with coordinate frame based commands.

Getting Started

Learn to use YP-Spur at the documentation pages.

Install

$ mkdir build
$ cd build
$ cmake ..
$ make
$ sudo make install

Purpose

YP-Spur is a mobile robot motion control software with coordinate frame based commands.

Install

$ mkdir build
$ cd build
$ cmake ..
$ make
$ sudo make install
